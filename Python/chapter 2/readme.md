## 2.1 Operators in Python
### Arithematics Operators:
```bash
a = 6
b = 8
print(a+b)           #Output: 14
```
Similarly for, <br>
Addition ```+``` , 
Subtraction ```-```, 
Multiplication ```*```, 
Exponentiation ```**```, 
Division ```/```, 
Floor Division ```//```, 
Modulus ```%```

### Assignment Operators:
```bash
a = 7
b = 5
b += 3      #increment the value of b by 3 and then assign it to b. (b=8)
print(b)
#Output: 8

b -= 4      #decrement the value of b by 4 and then assign it to b. (8-4=b=4)
print(b)
#Output: 4
```
